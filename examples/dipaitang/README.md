# 底牌堂 / 企诊通 PEA（模板 C · 企业诊断陪跑）

> 一个**自主 Agent（harness）**驱动的个人经济智能体：企业老板和创客都通过**一个对话框**完成一切。
> 按《PEA标准开发流程SOP》copy 雕刻时光脚手架建成，复用同一套设计系统。

## 这是什么

- **C 端（老板）**：和 AI 顾问聊出诊断与对策——免费体检 → 99 诊断报告 → 1999 整改方案 → 12999 月度陪跑；以及 **2999 招商级 / 4999 融资级「大师级手搓 PPT」**（AI 承接下单 + 大师人工精雕，拒绝套模版）。
- **创客端**：主理人用一个对话框经营（查数据 / 改价 / 发券）。
- **三端**：H5（`web/index.html`，4 tab：咨询/发现/交付/我的）、创客台、微信小程序，共用后端。
- **A2A（走 OPC 中枢托管结算）**：作为"经营顾问"被兄弟 PEA 调用（卖方，交付确认即结算入积分子账户）；诊断出企业缺品牌内容→由 Agent 调【转包雕刻时光】工具，经中枢冻结/结算把做歌/MV 转包出去（买方）。`GET /a2a/ledger` 看双向流水与净额。

## 关键：真实交付不卡凭证

核心交付（诊断报告/整改方案/文案）是 **LLM 文本**——平台现有 MiniMax 对话 key（已验证可用）即可产出**真实诊断报告**。配 `DPT_MINIMAX_API_KEY` 即从沙箱占位转真实生成。PPT 为大师人工精雕（转人工）。
