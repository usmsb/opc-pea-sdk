# 喵星球 PEA（模板 B · 宠物零售 / 上门服务 / 情感纪念）

> 一个**自主 Agent（harness）**驱动的个人经济智能体：铲屎官通过**一个对话框**完成一切。
> 按《PEA标准开发流程SOP》copy 底牌堂脚手架建成（第三个 PEA），第三套视觉主题。

## 这是什么

三大业务引擎，一个猫咪生活管家全包：
- **好物商城（高频）**：猫砂 / 猫粮 / 罐头 / 玩具，19.9–99 元，对话推荐即下单。
- **上门服务（高毛利）**：上门喂养 59 / 上门洗护 99 / 精致养护月卡 299 / 月度喂养卡 499。
- **宠物纪念 · 经纬度纪念碑（情感壁垒，核心）**：线上基础纪念册 99 / 经纬度线上纪念碑 299 / 实体碑代办 520。
  永久存照片视频生平 + 真实地图坐标 + 留言献花烛火 + 纪念日提醒。**别人没有的壁垒。**
- **裂变**：晒图返现（好物/服务/纪念按档返）+ **二级分佣**（一级 15% / 二级 5%，满 50 可提）。

## 关键：真实交付 + A2A

- **纪念生平/悼词**是 **LLM 文本**——平台现有 MiniMax 对话 key（已验证）即可写出**真实的宠物悼词**。配 `MXQ_MINIMAX_API_KEY` 即从沙箱占位转真实生成。
- **A2A 买方**：铲屎官想给毛孩子加一首专属纪念曲 → Agent 调【转包雕刻时光】，经 **OPC 中枢托管结算**把做歌转包出去（喵星球自己不做歌）。
- **A2A 卖方**：作为"宠物养护顾问"被兄弟 PEA 调用（`/a2a`），交付确认即结算入积分子账户。`GET /a2a/ledger` 看双向流水。

## 运行（沙箱：无 key 全功能可跑可测）

```bash
cd peas/miaoxingqiu
python3.12 -m uvicorn app.main:app --port 8097   # → http://127.0.0.1:8097
```

```bash

python3.12 tests/run_e2e.py   # 离线全漏斗 trace + 断言（好物+服务+纪念+纪念曲+返现）
```
